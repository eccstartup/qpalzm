How to use hackage mirror.

Most packages are installed by "cabal install pkg-name".

For a user, the config file is located at "~/.cabal/config".

Just replace

remote-repo: hackage.haskell.org:http://hackage.haskell.org/packages/archive

with

remote-repo: 60161:http://202.117.60.161/mirror/hackage/packages/archive

So, just add the link of this site to the config file, you can download haskell package locally.

Have fun!
